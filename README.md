# HandHistoryParser

Парсер истории рук из PokerStars.

## Структура проекта
	+parser/Parser.cs -- основной файл проекта. Содержит 2 публичных метода: ParseZipAndWriteOutput(string zipFilePath) | Parse(string handHistory)
	+resources -- здесь размещается архивный файл с историей раздач
	+utils -- вспомогательные классы и методы

## Использование
1. Поместите ваш архивный файл с историей раздач в папку `/resources`.
2. Запустите приложение. 
3. Проверьте папку `/parsed` для результатов.
4. Ознакомьтесь с логами в `/logs` для отладки и мониторинга.
